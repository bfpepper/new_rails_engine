# README

* Rials version
  5.0.0.1

* Getting Started
  `Git clone git@github.com:bfpepper/new_rails_engine.git`

  `bundle`

  `rake db:create db:migrate db:test:prepare`
  
  `rake load_data:all`

* How to run the test suite
  tests are in RSpec so just run `rspec`

* to boot the server
  `rails s`

* to get to the console
  `rails c`
